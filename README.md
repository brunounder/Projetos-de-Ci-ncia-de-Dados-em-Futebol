#  Análise de Dados: Ligas Europeias de Futebol (2008-2016)

Este projeto realiza uma análise estatística completa sobre o desempenho de times e ligas europeias. O objetivo é transformar dados brutos em insights visuais sobre o esporte mais popular do mundo.

##  Tecnologias Utilizadas
* **Linguagem:** Python
* **Banco de Dados:** SQLite (SQL para extração de dados)
* **Bibliotecas:** 
  * `Pandas` (Manipulação e limpeza)
  * `Plotly` (Gráficos interativos)
  * `Sqlite3` (Conexão com o banco de dados)
* **Ambiente:** Google Colab

## O que foi analisado?
1. **Ranking de Pontos:** Cálculo de 3 pontos por vitória para descobrir os times mais dominantes.
2. **Fator Casa:** Comparação de vitórias dentro e fora de casa por liga.
3. **Média de Gols:** Identificação das ligas mais ofensivas e defensivas da Europa.
4. **Evolução Temporal:** Como o comportamento das ligas mudou ao longo das temporadas.

##  Como executar o projeto
1. Baixe o dataset `database.sqlite` no Kaggle (link_aqui).
2. Abra o arquivo `.ipynb` no Google Colab.
3. Faça o upload do arquivo `.sqlite` na sessão.
4. Execute as células de código em ordem.
